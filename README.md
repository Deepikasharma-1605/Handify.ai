# ✍️ Handify AI

Handify AI is an AI-powered productivity application that transforms typed text into realistic handwritten notes. Users can simply type or paste their content, and the application automatically converts it into handwritten pages or notebooks. It can also generate presentation-ready PowerPoint slides, making it a useful tool for students, educators, and professionals to reduce manual work and save time.

---

## 🚀 Features

- 📝 Convert typed or pasted text into realistic handwritten notes.
- 📖 Generate handwritten pages with customizable notebook styles.
- 🎨 Multiple handwriting fonts and page templates.
- 📄 Export handwritten notes as PDF.
- 📊 Automatically generate PowerPoint presentations from text.
- ⚡ Fast and user-friendly interface.
- 💾 Download generated files for offline use.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/Handify-AI.git
```

### Move into the project directory

```bash
cd Handify-AI
```

### Create a virtual environment (Optional)

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Visit:

```
http://127.0.0.1:5000
```

## 📖 How It Works

1. Enter or paste your text into the editor.
2. Select a handwriting style and notebook template.
3. Click **Generate**.
4. Handify AI converts the text into realistic handwritten pages.
5. Download the handwritten notes as PDF or images.
6. Optionally generate a PowerPoint presentation from the same content.
